# Vector-Quant: High-Performance Market Backtesting Engine

A pure-NumPy engine built to process and analyze financial market data at scale. This project demonstrates advanced vectorization, broadcasting, and memory-mapping techniques without relying on Pandas.

## Setup
1. `pip install -r requirements.txt`
2. Run `python src/data_gen.py` to simulate market data.
3. Run `python src/engine.py` to execute the core analytics pipeline.
